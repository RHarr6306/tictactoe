A simple TicTacToe game made with React.js.

## Getting Started
Clone the repository & install required dependencies
```
$ git clone git@github.com:Le-SirH/tictactoe.git
$ npm install
$ yarn start
```

## Built With

* [React.js](https://reactjs.org/)
